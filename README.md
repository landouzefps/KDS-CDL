# Site Web KingDoom Squad

Un site web moderne et élégant pour KingDoom Squad, construit avec Next.js et Tailwind CSS.

## Technologies Utilisées

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion

## Installation

1. Clonez le repository
2. Installez les dépendances :
```bash
npm install
```

## Démarrage du Projet

Pour lancer le serveur de développement :

```bash
npm run dev
```

Le site sera accessible à l'adresse [http://localhost:3000](http://localhost:3000)

## Structure du Projet

- `/app` - Contient les pages et composants principaux
- `/public` - Fichiers statiques (images, etc.)
- `/components` - Composants réutilisables

## Fonctionnalités

- Design moderne et responsive
- Animations fluides avec Framer Motion
- Sections pour présenter l'équipe et les résultats
- Interface utilisateur intuitive
